# Output Templates

These templates define the locked, deterministic output formats for all KYC Analyst reports.

## Templates

### PDF_Report_Template.md
17-section professional PDF report template with:
- Title banner and customer details
- Executive summary
- Step 0 search results (all 5 searches)
- Decision gate analysis
- Risk assessment
- Compliance statement
- Approvals section

### Case_Folder_Structure.md
Locked folder structure specification with:
- 7 mandatory subfolders per case
- File naming conventions
- Organization rules

## Usage

These templates are automatically applied when generating reports. Do not modify the template structure -- only fill in the data fields.

**Version:** 1.0.0
